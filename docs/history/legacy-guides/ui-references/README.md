## 月见的综合结论

1. 当前只维护标准 Light 与 Dark：中性灰阶表面、系统蓝交互色、独立语义状态色。
2. 人物设定色、第三方主题和动态取色留到基础主题稳定之后，不进入当前视觉基线。
3. 控制中心使用单侧栏、主工作区和按需详情栏，不叠加 IDE 式双层导航。
4. 主工作区保持连续平面；卡片只用于真正独立、可重复或可浮出的对象。
5. Mica 用于窗口根层或导航背景，Acrylic 只用于菜单、对话框和临时浮层。
6. 所有按钮必须具备 hover、pressed、focus、disabled 和 pending 状态。
7. 分栏不等于画框：默认依靠表面色阶、对齐和留白，分割线只作为低对比辅助。
