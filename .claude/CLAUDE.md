# Glimmer Cradle Claude Code 入口

本文件是 Claude Code 的项目级薄入口，不保存项目事实。

开始任何任务前：

1. 阅读根目录 `AGENTS.md`。
2. 阅读 `docs/README.md`，按任务进入 Architecture、Reference、Guides 或 Roadmap。
3. 如可访问，阅读 `.codex/skills/glimmer-cradle/SKILL.md`，并按它的 reference 路由读取最少必要的 `.codex/skills/glimmer-cradle/references/`。

项目事实只在 `docs/` 与 `AGENTS.md` 中维护；agent 操作卡片只在 `.codex/skills/glimmer-cradle/references/` 维护。不要在 `.claude/` 下复制第二套架构、流程或反模式文档。
